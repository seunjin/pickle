# 🥒 Pickle Note Extension Features

Pickle Note 브라우저 익스텐션의 주요 기능과 사용 방법을 정리한 문서입니다.

## 1. 📝 텍스트 저장 (Save Text)
웹페이지에서 기억하고 싶은 문장을 발견했을 때 바로 저장할 수 있습니다.

- **사용법**:
    1. 웹페이지에서 텍스트를 드래그하여 선택합니다.
    2. 마우스 오른쪽 버튼을 클릭하여 Context Menu를 엽니다.
    3. **"📝 텍스트 저장하기"** 메뉴를 클릭합니다.
- **동작**:
    - 우측 상단에 오버레이 모달이 나타납니다.
    - 선택한 텍스트가 자동으로 입력되어 있습니다.
    - 추가 메모를 남기고 저장할 수 있습니다.

## 2. 🖼️ 이미지 저장 (Save Image)
웹상의 이미지를 손쉽게 저장하고 관리할 수 있습니다.

- **사용법**:
    1. 저장하고 싶은 이미지 위에서 마우스 오른쪽 버튼을 클릭합니다.
    2. **"🖼️ 이미지 저장하기"** 메뉴를 클릭합니다.
- **동작**:
    - 오버레이 모달에 해당 이미지가 미리보기로 표시됩니다.
    - 이미지에 대한 메모를 덧붙여 저장할 수 있습니다.

## 3. 🔖 북마크 (Bookmark)
현재 보고 있는 웹페이지 전체를 스크랩합니다.

- **사용법**:
    1. 페이지 빈 공간 어디서든 마우스 오른쪽 버튼을 클릭합니다.
    2. **"🔖 북마크"** 메뉴를 클릭합니다.
- **동작**:
    - 페이지의 메타데이터(Title, Description, Thumbnail, Favicon, Site Name)를 자동으로 추출합니다.
    - 깔끔한 카드 형태로 미리보기를 제공하며 저장할 수 있습니다.

## 4. 📷 화면 캡쳐 (Screen Capture)
원하는 영역을 직접 지정하여 캡쳐하고 저장합니다.

- **사용법**:
    - **Context Menu**: 마우스 우클릭 -> **"📷 캡쳐하기"** 클릭.
    - **단축키**: `Shift` + `Cmd` + `E` (Mac 기준) / `Shift` + `Ctrl` + `E` (Windows 기준).
- **동작**:
    1. 화면이 캡쳐 모드로 전환되며 마우스 커서가 십자 모양으로 변합니다.
    2. 원하는 영역을 드래그하면 즉시 캡쳐됩니다.
    3. 캡쳐된 이미지가 오버레이 모달에 표시되며 저장할 수 있습니다.
    - *참고: ESC 키를 누르면 캡쳐가 취소됩니다.*

## 5. 🚀 UX/UI 특징 (Shadow DOM Overlay)
사용자 경험을 해치지 않기 위해 세심하게 설계되었습니다.

- **Non-Intrusive**: 기존의 Side Panel 방식에서 **Floating Overlay** 방식으로 변경하여, 원본 페이지의 레이아웃(Layout Shift)을 전혀 방해하지 않습니다.
- **Style Isolation**: Shadow DOM 기술을 적용하여, 웹페이지의 CSS가 익스텐션 UI에 영향을 주거나 반대로 익스텐션 스타일이 웹페이지를 망가뜨리지 않습니다.
- **Independent State**: 여러 탭을 동시에 열고 사용해도 각 탭의 데이터가 섞이지 않고 독립적으로 유지됩니다.
